# 메디컬 마케팅 사업 컨텍스트

> ARK Point 탐색 중인 신사업. 현재 베러미의원 레퍼런스로 검증 진행 예정.
> 최종 업데이트: 2026-03-12

---

## 사업 한 줄 정의

**피부과 병원의 검색 기반 환자 유입을, 의사의 목소리가 담긴 콘텐츠 시스템으로 설계하는 에이전시**

- 포지션: "블로그 대행"이 아닌 **검색 퍼널 설계**
- 목적: 신규 환자(신환) 확보
- AI = 생산성 도구 (사업 본질은 검색 퍼널 설계)

---

## 왜 지금인가 (Why Now)

- AI로 콘텐츠 생산 원가 30~40% 절감 가능
- 기존 에이전시 대비 저렴 + 신환 측정 가능 = 차별화
- 기존 대행사 문제: 키워드 전략 없음, 성과 측정 없음, 원장 목소리 없음
- 원장 불만 언어: "돈은 쓰는데 뭘 얻는지 모르겠다"

---

## 타깃 고객

**조건 (4가지 동시 충족)**

1. 월 매출 1억 이상 피부과
2. 마케팅 월 400만원 이상 지출 (블로그만 150만원 이상)
3. 현재 에이전시 사용 중 또는 경험 있음
4. 내부 마케팅 인력 없음

**진료과 우선순위**: 피부과 → 성형외과 → 치과 → 한의원 → 정형외과

---

## 핵심 차별화 3가지

### 1. 신환 측정 시스템
- 블로그 포스팅 → 전용 상담번호 → 인바운드 콜 집계 → 병원 CRM 대조
- 경쟁사 리포트: 조회수/CTR | ARK Point 리포트: 키워드 순위 + 블로그 경유 콜 + 예약 전환 + **신환 추정**
- 이게 없으면 6~12개월 후 계약 해지 (성과 귀속 문제)

### 2. Doctor Insight Capture (의사 목소리)
- 원장 월 30~35분 참여로 진정성 있는 콘텐츠 생산
- 수집 방식: ① 3~5분 음성 노트 → 텍스트 변환 ② 월 30분 인터뷰 ③ 카톡 단문 질문
- STICC 컨텍스트 패키지 (병원별 IP): Situation / Task / Intention / Concern / Calibration

### 3. 키워드 맵 100개 (병원 1개 기준)
- Core 20개: 지역 + 시술 (대구 리쥬란, 대구 쥬베룩) → 전환율 높음
- Mid 40개: 시술 정보 (리쥬란 효과, 쥬베룩 가격)
- Long-tail 40개: 환자 질문 (리쥬란 몇 번 맞아야 하나요)
- **온보딩 첫 작업 = 키워드 맵 100개 설계**

## 콘텐츠 시스템

**월 16건 구성**

| 유형 | 건수 | 역할 |
|---|---|---|
| 시술 콘텐츠 | 6 | Core + Mid 키워드 |
| 피부 고민 콘텐츠 | 4 | Stage1 진입 |
| FAQ 콘텐츠 | 4 | Long-tail 트래픽 |
| 원장 브랜딩 | 2 | 차별화 |

**환자 검색 3단계**: 문제 인식 → 시술 탐색 → 병원 선택

**이미지 정책**: 병원 실사진 50장 이상 확보. AI 생성 인물 이미지 사용 금지.

## 가격 구조

| 플랜 | 월 가격 | 포스팅 | 순이익 |
|---|---|---|---|
| BASIC | 150만원 | 8건 | — |
| PRO | 230만원 | 16건 | 약 75만원 |
| GROWTH | 350만원 | 16건+ | — |

**PRO 원가**: Editor 120만 + AI 툴 25만 + 기타 10만 = 155만 → **순이익 75만원**

**수익 시뮬레이션**: 클라이언트 5건 → 375만원 / 10건 → 750만원

## 조직 구조 (초기)

- HS: 세일즈 + 고객관리 + 전략
- Medical Editor 1명: 키워드맵 + AI 초안 + 의료법 체크 + 블로그 업로드 + KPI 수집
  - 채용 기준: 간호사 배경 또는 의료 도메인 이해 있는 인력

---

## 강점 / 약점

**강점**: Warm market (SKY 전문직 커뮤니티 의사 10명+), 세일즈 능력, AI 활용, 크로스보더 경험

**약점**: SEO 실전 경험 부족, 의료 도메인 이해 부족, 에이전시 운영 경험 없음

**해결**: 베러미의원 3개월 테스트 레퍼런스 확보 → Editor 채용

---

## 레퍼런스 현황

- **베러미의원**: 현재 라랩 협업으로 보유한 레퍼런스. 첫 3개월 실험 대상.

---

## 90일 실험 계획

| Month | 작업 |
|---|---|
| 1 | 병원 실사진 50장, 전용 추적번호 세팅, 키워드맵 100개, 포스팅 16건, STICC 컨텍스트 패키지 |
| 2 | 키워드 순위 추적, 포스팅 16건, 인바운드 콜 수집 |

**성공 판단 기준**: Core 키워드 순위 상승 + 블로그 경유 인바운드 콜 발생

---

## 장기 로드맵

- Phase 1: 콘텐츠 에이전시 (목표 클라이언트 10~15건, 매출 2,300만원)
- Phase 2: 성과 기반 모델 (기본비 + 신환당 수익)
- Phase 3: Clinic SaaS (병원 직접 사용 / 에이전시 B2B 판매)
