# 사업 한 줄 정의

**피부과 병원의 검색 기반 환자 유입을, 의사의 목소리가 담긴 콘텐츠 시스템으로 설계하는 에이전시**

"블로그 대행"을 넘어 신규 환자를 유치하는 **검색 퍼널 설계 회사**다.

AI는 생산성 도구이고, 콘텐츠는 채널이며, 목적은 **신규환자 확보**다.

# 1. 왜 지금인가

## 시장 구조의 문제

병원 → 마케팅 대행사 → 콘텐츠 외주

대부분 대행사의 실제 현실

- 키워드 전략 없이 글 발행
- 의료 전문성 없는 콘텐츠
- 성과 측정 없이 리포트만 발행
- 원장 목소리 없는 광고성 글
- **원장님의 신뢰를 받지 못함**

결과

→ 원장은 **“돈은 쓰는데 뭘 얻는지 모르겠다”**고 느낀다.

## 기회

AI로 콘텐츠 생산 원가가 내려갔다.

기존 에이전시와 같은 품질을 **30~40% 저렴하게 제공하면서**

추가로 제공 가능

- 신환 측정
- 원장 목소리 기반 콘텐츠

# 2. 타깃 고객 — 누구

## 프라이머리 타깃

다음 조건을 동시에 충족하는 개원의

- 월 매출 1**억 이상의 피부과 부터 시작해서 확장**
- 마케팅에 월 400, 블로그에만  월 **150만원 이상 지출 중**
- 에이전시 경험 있거나 현재 사용 중
- “블로그 중요한 건 아는데 직접 할 시간 없다”
- 내부 마케팅 인력 없음

## 왜 이 조건인가

조건 미달 시 문제

- 마케팅 미경험 병원 → 교육 비용 과다
- 매출 1억 미만 → 예산 없음
- 내부 담당자 있는 병원 → 우리 필요 없음

## 우선순위 진료과

피부과 → 성형외과 → 치과 → 한의원 → 정형외과

기준

- 콘텐츠 소비량
- 환자 온라인 탐색 비율

# 3. 고객의 문제 — 어떤 문제

## 핵심 문제

**에이전시는 콘텐츠를 팔고, 원장이 원하는 것은 신환이다.**

이 둘이 연결되어 있지 않다.

## 레이어별 문제

| 문제 | 원장의 실제 언어 |
| --- | --- |
| 측정 불가 | 리포트 봐도 병원이 잘 되는지 모르겠어 |
| 진정성 부재 | 내가 쓴 것도 아닌데 환자가 믿을까 |
| 시간 부족 | 진료하면 하루가 끝나, 글 쓸 시간 없어 |
| 성과 개선 불가 | 에이전시가 문제는 아는데 못 고치고 있어 |

## 숨겨진 위험 — 성과 귀속 문제

병원 마케팅의 가장 흔한 실패 패턴

환자 늘었다 → 병원이 잘해서

환자 줄었다 → 마케팅이 문제

결과

→ **6~12개월 후 계약 해지**

해결 방법

**마케팅 기여도를 숫자로 증명해야 한다**

신환 측정은 선택이 아니라 **생존 조건**이다.

# 4. 피부과 환자 검색 구조

환자의 의사결정은 **3단계 검색 구조**로 이루어진다.

## Stage 1 — 문제 인식

환자가 피부 문제를 인식

→ 검색

예시

- 모공 넓어짐
- 피부 탄력
- 피부 재생

콘텐츠 역할

- 문제 설명
- 원인 분석

목적

**첫 접촉 + 신뢰 시작**

---

## Stage 2 — 시술 탐색

환자가 해결 방법을 찾는 단계

검색 예시

- 리쥬란 효과
- 쥬베룩 후기
- 울쎄라 효과

콘텐츠 역할

- 시술 설명
- 효과
- 부작용
- 시술 비교

목적

**신뢰 형성**

---

## Stage 3 — 병원 선택

병원을 결정하는 단계

검색 예시

- 대구 리쥬란
- 대구 피부과 추천

콘텐츠 역할

- 병원 소개
- 의료진
- 케이스
- 가격

목적

**예약 직전 전환**

---

## 중요한 구분

블로그 = **신뢰 채널**

예약 발생 채널

- 네이버 플레이스
- 카카오 상담

블로그 역할

신뢰 형성 → 플레이스/카카오에서 예약 전환

블로그 성과

- 키워드 노출
- 블로그 트래픽
- 블로그 경유 인바운드

시술명 + 정보

예

- 쥬베룩 가격

역할

시술 탐색 단계 포착

신뢰 형성

---

## Long-tail Keywords (40개)

환자 실제 질문 기반

예

- 리쥬란 몇 번 맞아야 하나요

역할

- FAQ 콘텐츠
- 블로그 트래픽 증가
- 체류시간 증가

# 7. Doctor Insight Capture

AI 콘텐츠만으로는 차별화가 없다.

**의사의 목소리가 핵심**

원장 참여 시간

월 **30~35분**

## 수집 방식

### 1. 음성 노트

3~5분 음성 메모

예

“오늘 리쥬란 환자가 이런 질문을 했는데…”

→ 텍스트 변환 → 콘텐츠 소재

### 2. 월 인터뷰

월 1회 30분

- 최근 환자 고민 TOP3
- 인상 깊은 케이스
- 관심 시술 트렌드

### 3. 카톡 질문

편집자가 질문 1개

예

“리쥬란 시술 전에 환자들이 가장 걱정하는 건?”

원장 답변 → 콘텐츠 핵심
